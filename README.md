# Forecasting energy consumption over time
## Project description
The main objective of this project is to analyze a time series and apply different techniques in order to be able to make predictions about the energy consumption over time based on historical data. Throughout the project, I'll learn how to apply different techniques, and use a combination of some, in order to make the data series stationary. In this process, I will learn that some techniques will achieve stationary but in return some data will be lost and we need to take that into account when selecting the techniques. Also, that sometimes one technique will not be enough to achieve stationary and more than one technique will be required.
Finally, when stationary is reached, I will use SARIMA model to forecast the data and execute some metrics to evaluate the results.

## Goal
Analyze and prepare a time series to forecast energy consumption data over time.

## Content
* Exploratory Data Analysis (EDA)
* Transforming Non-Stationary Time Series
* Forecasting with SARIMA model
* Results analysis and conclusion
